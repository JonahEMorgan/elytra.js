Text Consumer

The purpose of a text consumer is to shorten a common pattern. Often an element's text will need to follow the data model, so a callback to do this is built in to every element.

      
<p id="text"><p>
<p id="count"><p>
      
    
      
elytra.add("model", {
  text: "Hello World!",
  count: 0
});
elytra.model.text.watcher = document.find.text.text;
elytra.model.count.watcher = document.find.count.text;
